comenzile de la tastatura ale clientului sunt:

logout / exit - deconectarea si inchiderea clientului.
submit / send - trimiterea fisierelor sursa serverului pentru a fi corectate.
time - cererea afisarii timpului ramas la dispozitie pentru terminarea problemelor.
rez - cererea trimiterii fisierului cu rezultatele.
info / basic info - cererea retrimiterii informatiilor referitoare la concurs ( nr probleme, numele acestora, timpul avut la dispozitie etc.)
    basic info nu mai trimite si fisierele .cpp .in si .out ale problemelor inca o data.
    info trimite si aceste fisiere.
